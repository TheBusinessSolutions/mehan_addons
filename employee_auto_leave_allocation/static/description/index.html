Automatic Leave Allocation

You can allocate leave to the newly created employee automatically.

As you might be knowing that the default Odoo functionality not allows to generate Approved leaved that by company so using this module you can assign or allocate any pre defined leaves to your newly added employee when you create new employee.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

Go to General Settings --> Employees --> Set True Automatically Leave Allocate and select Leave Type for which leave type you want to allocate automatically leaves to newly created employee.

Step-By-Step Guidelines

Once You set Configuration and create Employee at time by company mode leave allocate to new employee. or you can see Left Leaves from Employee form view

Once New Employee Created you can see allocated Leave from Leaves menu.
